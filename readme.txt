Pyqt версии 5, скачать отсюда:
   http://sourceforge.net/projects/pyqt/files/PyQt5/PyQt-5.6/PyQt5-5.6-gpl-Py3.5-Qt5.6.0-x32-2.exe/download 
Модуль для python: dxfwrite, скачать отсюда:
   http://pypi.python.org/packages/b5/23/6f09201d1bc506b3bf36e381ac6d5886714e6e7b84ad54bb961d25411c1b/dxfwrite-1.2.1.zip#md5=8a94c11415323b5369e0e75ac7cd7fd0

Запускаем по очереди:
    python-3.5.4.exe (рекомендую устаноновить python в папку c:\python35)

переходим в распакованную папку и выполняем установку:

    Для программистов

Чертеж генерируется в один файл, но по слоям.

Каждый слой - одна деталь.

Деталь может состоять из примитивов:
    линий, дуг, окружностей, прямоугольников, овалов.

Каждого вида примитивов может быть до 49 штук, в одной детали.

Слои рспределены для примитивов так:
     c 1-й по 49-ю помещаются в слой SIDER (правая стенка-деталь модели)
     c 50-й по 99-ю помещаются в слой SIDEL (левая стенка-деталь модели)
     c 100-й по 149-ю помещаются в слой TOP (верняя стенка-деталь модели)
     c 150-й по 199-ю помещаются в слой BOTTOM (нижняя стенка-деталь модели)
     c 200-й по 249-ю помещаются в слой FACE (передняя стенка-деталь модели)
     c 250-й по 300-ю помещаются в слой ANFACE (задняя стенка-деталь модели)
