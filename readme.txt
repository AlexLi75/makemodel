    Установка

Необходим Python версии 3. скачать отсюда:
   http://www.python.org/ftp/python/3.5.4/python-3.5.4.exe 
Pyqt версии 5, скачать отсюда:
   http://sourceforge.net/projects/pyqt/files/PyQt5/PyQt-5.6/PyQt5-5.6-gpl-Py3.5-Qt5.6.0-x32-2.exe/download 
Модуль для python: dxfwrite, скачать отсюда:
   http://pypi.python.org/packages/b5/23/6f09201d1bc506b3bf36e381ac6d5886714e6e7b84ad54bb961d25411c1b/dxfwrite-1.2.1.zip#md5=8a94c11415323b5369e0e75ac7cd7fd0 
Для пользователей Windows:
Запускаем по очереди:
    python-3.5.4.exe (рекомендую устаноновить python в папку c:\python35)
    PyQt5-5.6-gpl-Py3.5-Qt5.6.0-x32-2.exe  
распаковываем архив: dxfwrite-1.2.1.zip

переходим в распакованную папку и выполняем установку:

python setyp.py install

Для работы программы всё готово.

Запускаем python c:\makemodel\mmodel.py
